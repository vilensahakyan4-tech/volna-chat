# VOLNA

Случайный видеочат на WebRTC без регистрации и внешних зависимостей.

## Запуск

```powershell
npm start
```

Откройте `http://localhost:3000`. Для проверки видеосвязи откройте адрес в двух вкладках или на двух устройствах. В продакшене обязательно используйте HTTPS и добавьте собственный TURN-сервер.
